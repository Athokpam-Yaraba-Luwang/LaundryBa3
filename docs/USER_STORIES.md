# User Stories

## Customer
1. **Registration**: As a new user, I want to register with my phone number so I can use the service.
2. **View Orders**: As a customer, I want to see my active and past orders with their status so I know when my laundry is ready.
3. **Offers**: As a customer, I want to see available offers and redeem codes so I can save money.
4. **Feedback**: As a customer, I want to rate my experience so the business can improve.

## Business Admin
1. **Intake**: As an admin, I want to take a photo of laundry items and have AI detect them so I can quickly create orders.
2. **Verification**: As an admin, I want to verify and correct AI detections to ensure accuracy.
3. **Dashboard**: As an admin, I want to see real-time revenue and order status to manage operations.
4. **Analytics**: As an admin, I want to run analysis on revenue and feedback to make data-driven decisions.
